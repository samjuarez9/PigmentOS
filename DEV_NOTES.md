# PigmentOS Development Notes & Changelog

## Overview
This document tracks the "bells and whistles" and quality-of-life improvements added to the PigmentOS dashboard.

## Changelog

### [Date: 2025-11-29]
- **Feature:** Auto-Scroll Button in Intel Feed Header.
    - *Details:* Added a button that toggles between "AUTO-SCROLL" (Green) and "HOVER PAUSE" (Yellow) based on hover state. Synced with the scroll engine.
- **Feature:** Custom Static Favicon.
    - *Details:* Updated to use user-provided pixel art 'P' logo.
- **Feature:** Local Storage Persistence.
    - *Details:* Dashboard now remembers the selected ticker across page reloads.
- **Feature:** Cyber Watchdog Connection Monitor.
    - *Details:* Added a real-time latency monitor with a "system diagnostics" aesthetic.
